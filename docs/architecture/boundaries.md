# Boundaries

CLECMAS maintains these boundaries:

- AI-assisted rules intelligence is required.
- AI authority is prohibited.
- Deterministic evaluation is authoritative.
- Human-approved rule versions control compliance status.
- Reviewed evidence controls completion credit.
- Alerts use deterministic trigger data.
- Licensed professionals retain final submission, payment, signature, certification, attestation, and penalty-of-perjury responsibility.
- Codex remains closed.
